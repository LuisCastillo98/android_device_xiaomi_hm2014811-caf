Xiaomi Redmi 2
==============

The Xiaomi Redmi 2 (codenamed _"HM2014811"_) is a high-end smartphone from Xiaomi.

It was announced in September 2014.

Memory  | 1GB RAM
Shipped Android Version | 4.4.4
Storage | 8GB
Battery | 2300 mAh
Display | 4.7" 1280 x 720 px
Camera  | 8MPx, LED Flash
